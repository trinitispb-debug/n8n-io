# 🚀 Быстрый запуск workflow в уже установленном n8n

## 1. 🔌 Запустите n8n

```bash
# Если установлен через npm
n8n start

# Если используете Docker
docker run -it --rm --name n8n -p 5678:5678 -v $(pwd)/n8n-data:/home/node/.n8n n8nio/n8n

# Если n8n уже запущен - пропустите этот шаг
```

## 2. 🌐 Откройте n8n в браузере

Перейдите на: **http://localhost:5678**

## 3. 📥 Импортируйте workflow

### Способ A: Импорт файла (Рекомендуется)
1. **Нажмите кнопку "+"** в левом верхнем углу
2. **Выберите "Import from file"**
3. **Загрузите файл** `n8n_workflow_examples.json`
4. **Выберите workflow** для импорта (импортируйте все 3)

### Способ B: Копирование JSON
1. **Создайте новый workflow** (кнопка "+")
2. **Нажмите на три точки** ⋯ в правом верхнем углу
3. **Выберите "Import from Clipboard"**
4. **Скопируйте JSON** из файла `n8n_workflow_examples.json`
5. **Вставьте и нажмите "Import"**

## 4. 🔧 Настройте основные параметры

### В каждом workflow замените:

#### HTTP Request ноды (Битрикс24):
```
БЫЛО: "url": "https://your-bitrix24.bitrix24.ru/rest/1/webhook_key/"
СТАЛО: "url": "https://ВАШ-ДОМЕН.bitrix24.ru/rest/1/ВАШ_КЛЮЧ/"
```

#### Google Sheets ноды:
```
БЫЛО: "sheetId": "your-sheet-id"
СТАЛО: "sheetId": "ВАШ_ID_ТАБЛИЦЫ"
```

#### Telegram ноды:
```
БЫЛО: "chatId": "your-chat-id"
СТАЛО: "chatId": "ВАШ_CHAT_ID"
```

## 5. ✅ Активируйте workflow

1. **Переключите тумблер "Active"** в правом верхнем углу каждого workflow
2. **Убедитесь, что индикатор стал зеленым**

## 6. 🧪 Протестируйте

### Получите URL webhook:
1. **Откройте workflow** "Inventory Reserve Main Workflow"
2. **Кликните на ноду "Webhook"**
3. **Скопируйте "Production URL"**

### Отправьте тестовый запрос:
```bash
curl -X POST "ВАШ_WEBHOOK_URL" \
  -H "Content-Type: application/json" \
  -d '{
    "product_id": "123",
    "quantity": 2,
    "customer_id": "456"
  }'
```

## 🎯 Минимальная настройка для тестирования:

Если хотите быстро протестировать без всех интеграций:

1. **Импортируйте только основной workflow**
2. **Отключите ноды** Google Sheets и Telegram (кликните → Disable)
3. **Оставьте только:** Webhook → Set → HTTP Request → Response
4. **Активируйте workflow**
5. **Тестируйте через webhook**

## 🔍 Проверка работы:

- **Executions** → посмотрите историю выполнений
- **Зеленые галочки** = успешно
- **Красные крестики** = ошибки

---

## 📞 Если что-то не работает:

1. **Проверьте статус workflow** (должен быть Active/зеленый)
2. **Посмотрите Executions** на ошибки
3. **Убедитесь, что n8n запущен** на порту 5678
4. **Проверьте, что webhook URL доступен**

**Готово!** Ваш workflow запущен и готов к работе! 🎉
